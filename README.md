## Para rodar a aplicação

- Para instalar os pacotes tanto na pasta backend, quanto nas pastas frontend
    
        npm install --force
    
- Para iniciar a aplicação backend
    
        npm run dev

- Para iniciar o frontend
        
        npm run dev
